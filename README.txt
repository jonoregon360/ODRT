ODRT - OneDrive Renaming Tool
This tool is designed to remove illegal characters from files and folders inside of a 
OneDrive sync folder, allowing these files to sync properly. Changes are logged
in a text file located at the root of the sync folder.

Illegal characters:
 # % * : < > ? / \ | { } ~ & ^
Notes:
- & is replaced with 'and', all other characters are replaced with a space
